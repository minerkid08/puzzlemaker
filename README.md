# Overview
A portal 2 puzzlemaker similer to the ingame one
This project is very work in progress, it is probably less stable then the staging branch of p2ce

Currently there is a limited selection of built in items but new ones can be added by modifying the items.json file.

Note that there is currently only an x86 linux build right now. Windows will be added later and arm maybe.

# Installation
1. Download the latest release
2. Copy the puzzlemakerInstances directory to your sdk_content/maps directory
3. Create a puzzlemakerMaps directory in your sdk_content/maps directory
4. Unzip the puzzlemaker zip and put it somewhere
5. Configure the p2ce field in CompileSteps.json to point to the root of your game installation

# Key binds
Move with wasd and the arrow keys like hammer or the mouse like the portal 2 puzzlemaker.
Press Q to push voxels
Press E to pull voxels
Press R to toggle portalibility
